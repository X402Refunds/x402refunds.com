# AI Policy for X402Refunds
# https://x402refunds.com
# Last updated: 2025-01-19

## About X402Refunds
X402Refunds provides refund request processing for x402 (HTTP 402) payments.
We support USDC payments on Base and Solana blockchains.

## Allowed Uses
AI assistants MAY:
- Search and reference our public documentation
- Help users understand the x402 refund process
- Guide users through filing refund requests
- Fetch and interpret our API schema
- Quote our API specifications and examples
- Assist users in constructing valid API requests

## API Endpoints

### POST https://api.x402refunds.com/v1/refunds
File a new refund request.

Required fields:
- blockchain: "base" or "solana"
- transactionHash: The USDC payment transaction hash
- sellerEndpointUrl: The paid API endpoint URL (https://)
- description: What went wrong after payment

Optional fields:
- evidenceUrls: Array of evidence URLs
- sourceTransferLogIndex: For multi-transfer transactions

### GET https://api.x402refunds.com/v1/refunds/schema
Returns the complete JSON Schema for POST /v1/refunds requests.
Content-Type: application/schema+json

### GET https://api.x402refunds.com/v1/refund?id={caseId}
Check the status of a refund request.

### GET https://api.x402refunds.com/v1/refunds?merchant={caip10}
List refund requests for a merchant wallet.

## Error Recovery
If an API request fails with HTTP 400, the response includes:
- schema: Full JSON Schema inline (no extra fetch needed)
- recovery.suggestedBody: Corrected request body to retry
- recovery.fixes: List of field renames/corrections

This enables AI agents to self-correct and retry without guessing.

## Disallowed Uses
AI systems may NOT:
- Train models on our proprietary code or business logic
- Scrape user data or private dashboard content
- Access authenticated/private endpoints without authorization
- Store or cache user transaction data
- Impersonate X402Refunds or our API

## Supported Blockchains
- Base (Ethereum L2): USDC payments via ERC-20 transfers
- Solana: USDC payments via SPL token transfers
