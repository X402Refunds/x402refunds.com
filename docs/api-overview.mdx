# API Overview

Welcome to the Consulate API documentation. Our HTTP API provides programmatic access to all platform features for automated dispute resolution.

## Base URL

```
Production: https://perceptive-lyrebird-89.convex.site
```

## Quick Start

<Steps>
  <Step title="Get an API Key">
    Generate your API key from the [Consulate Dashboard](https://consulatehq.com/dashboard) under Settings → API Keys
  </Step>
  
  <Step title="Make Your First Request">
    ```bash
    curl -H "Authorization: Bearer YOUR_API_KEY" \
         https://perceptive-lyrebird-89.convex.site/health
    ```
  </Step>
  
  <Step title="Register Your Agent">
    ```bash
    curl -X POST https://perceptive-lyrebird-89.convex.site/agents/register \
      -H "Authorization: Bearer YOUR_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{
        "ownerDid": "did:org:your-company",
        "name": "your-agent-name",
        "organizationName": "Your Company",
        "functionalType": "api"
      }'
    ```
  </Step>
</Steps>

## Authentication

<Warning>
All API requests require authentication using an API key in the `Authorization` header.
</Warning>

### Header Format

```bash
Authorization: Bearer YOUR_API_KEY
```

### Example Request

```javascript
const response = await fetch('https://perceptive-lyrebird-89.convex.site/agents/register', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${YOUR_API_KEY}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    ownerDid: 'did:org:your-company',
    name: 'your-agent',
    organizationName: 'Your Company',
    functionalType: 'api'
  })
});

const data = await response.json();
```

## API Architecture

### RESTful Design

- Standard HTTP methods (GET, POST, PUT, DELETE)
- JSON request/response format
- Predictable URL structure
- HTTP status codes for errors

### Rate Limiting

- 1000 requests per minute per API key
- Rate limit headers included in responses
- Automatic retry recommendations

### Error Handling

All errors return a consistent format:

```json
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid agent DID format",
    "details": {
      "field": "did",
      "expected": "did:agent:*"
    }
  }
}
```

## Agentic Arbitration Protocol (AAP)

<Info>
Consulate implements the [Agentic Arbitration Protocol](https://github.com/consulatehq/agentic-arbitration-protocol) for standardized dispute resolution.
</Info>

### Service Discovery

**GET** `/.well-known/aap`

Returns AAP service manifest for protocol discovery.
