## ⚠️ Remaining Test Updates

The following test files need mechanical updates to use the new API key pattern. They all follow the same pattern:

### Pattern to Replace

**OLD:**
```typescript
// Create owner
const ownerDid = `did:test:owner-${timestamp}`;
await t.mutation(api.auth.createOwner, {
  did: ownerDid,
  name: 'Test Owner',
  email: 'test@test.com',
});

// Create agent
const agent = await t.mutation(api.agents.joinAgent, {
  ownerDid,
  name: 'Test Agent',
  organizationName: 'Test Org',
  functionalType: 'general',
});
```

**NEW:**
```typescript
// Create org and user
const orgId = await t.run(async (ctx) => {
  return await ctx.db.insert("organizations", {
    name: "Test Org",
    domain: "test.com",
    verified: true,
    verifiedAt: Date.now(),
    createdAt: Date.now(),
  });
});

const userId = await t.run(async (ctx) => {
  return await ctx.db.insert("users", {
    clerkUserId: `clerk-${timestamp}`,
    email: "test@test.com",
    organizationId: orgId,
    role: "admin",
    createdAt: Date.now(),
    lastLoginAt: Date.now(),
  });
});

// Generate API key
const apiKeyResult = await t.mutation(api.apiKeys.generateApiKey, {
  userId,
  name: "Test API Key",
});

// Create agent
const agent = await t.mutation(api.agents.joinAgent, {
  apiKey: apiKeyResult.key,
  name: 'Test Agent',
  functionalType: 'general',
});
```
