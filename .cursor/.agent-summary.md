### 1. Schema Changes (`convex/schema.ts`)

### 2. Backend Implementation

#### Created `convex/apiKeys.ts`
- `generateApiKey` - Generate new API keys (csk_live_* format)

- `revokeApiKey` - Revoke API keys with authorization

- Auto-creates owner DIDs for organizations

## 💡 Developer Experience

**Before:**
```typescript
// Complex: Manual dashboard creation, registration tokens, signatures
await ctx.mutation(api.agents.createOrgAgent, {
  userId, name, organizationName, functionalType
})
// Then: Save registration token, call addAgentPublicKey with signature...
```

**After:**
```bash
# Simple: One API call with your API key
curl -X POST https://api.consulate.io/agents/register \
  -H "Authorization: Bearer csk_live_abc123..." \
  -d '{"name": "My Agent"}'
```

Or with SDK:
```typescript
const consulate = new Consulate(process.env.CONSULATE_API_KEY)
const agent = await consulate.agents.register({ name: "My Agent" })
```

**Result:** 10x simpler for developers, infinitely scalable, zero manual intervention. 🚀
