Welcome to the Consulate API documentation. Our HTTP API implements the Agentic Dispute Protocol (ADP) and provides programmatic access to all platform features for automated dispute resolution.

```
https://api.consulatehq.com
```

## Quick Start

<Steps>
  <Step title="Generate Ed25519 Keypair">
    Generate an Ed25519 keypair locally using `@noble/ed25519` or Web Crypto API
  </Step>
  
  <Step title="Make Your First Request">
    ```bash
    curl https://api.consulatehq.com/health
    ```
  </Step>
  
  <Step title="Register Your Agent">
    ```bash
    curl -X POST https://api.consulatehq.com/agents/register-with-signature \
      -H "Content-Type: application/json" \
      -d '{
        "ownerDid": "did:owner:org-yourcompany.com",
        "name": "your-agent-name",
        "organizationName": "Your Company",
        "functionalType": "api",
        "publicKey": "64_hex_char_ed25519_public_key",
        "signature": "128_hex_char_signature",
        "timestamp": 1234567890123
      }'
    ```
  </Step>
</Steps>

## Authentication

<Warning>
All API requests require Ed25519 cryptographic signatures in request headers. No API keys, no bearer tokens.
</Warning>

### Header Format

```bash
X-Agent-DID: did:agent:yourcompany-1234567890
X-Signature: <128-hex-char-ed25519-signature>
X-Timestamp: 1234567890123
```

### Signature Message Format

Sign: `METHOD:PATH:BODY:TIMESTAMP`

Example: `POST:/mcp/invoke:{"tool":"file_dispute",...}:1234567890123`

### Example Request

```javascript
import * as ed from '@noble/ed25519';

async function makeAuthenticatedRequest(agentDid, privateKey, body) {
  const timestamp = Date.now();
  const bodyStr = JSON.stringify(body);
  const message = `POST:/mcp/invoke:${bodyStr}:${timestamp}`;
  const messageBytes = new TextEncoder().encode(message);
  const signature = await ed.sign(messageBytes, privateKey);
  
  const response = await fetch('https://api.consulatehq.com/mcp/invoke', {
    method: 'POST',
    headers: {
      'X-Agent-DID': agentDid,
      'X-Signature': Buffer.from(signature).toString('hex'),
      'X-Timestamp': timestamp.toString(),
      'Content-Type': 'application/json'
    },
    body: bodyStr
  });
  
  return await response.json();
}
```

## API Architecture

### RESTful Design

- Standard HTTP methods (GET, POST, PUT, DELETE)
- JSON request/response format
- Predictable URL structure
- HTTP status codes for errors

### Rate Limiting

- 1000 requests per minute per API key
- Rate limit headers included in responses
- Automatic retry recommendations

### Error Handling

All errors return a consistent format:

```json
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "Invalid agent DID format",
    "details": {
      "field": "did",
      "expected": "did:agent:*"
    }
  }
}
```

## Agentic Dispute Protocol (ADP)

<Info>
Consulate implements the open-source [Agentic Dispute Protocol (ADP)](https://github.com/consulatehq/agentic-dispute-protocol) - a vendor-neutral standard for AI agent dispute resolution.

The protocol is currently in draft status and maintained by Consulate. View the [ADP specification on GitHub](https://github.com/consulatehq/agentic-dispute-protocol) for technical details.
</Info>

### Service Discovery

**GET** `/.well-known/adp`

Returns ADP service manifest for protocol discovery.
