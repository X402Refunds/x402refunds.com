## Overview

If you operate a platform where agents interact (payments, API gateway, marketplace), you can provide built-in dispute resolution by integrating with Consulate.

**Your users get:** Automatic dispute filing when transactions fail  
**You get:** Reduced support burden, better merchant/customer trust  
**Integration time:** 30 minutes to 2 hours depending on pattern

## Integration Patterns

<Tabs>
  <Tab title="🏆 Proxy Pattern (Recommended)">
    ### Best for: Seamless user experience
    
    **How it works:** Your MCP server wraps Consulate tools with your branding.
    
    ```typescript Your MCP Server
    // Example: Locus payment platform
    export const MCP_TOOLS = [
      // Your native tools
      {
        name: "locus_process_payment",
        description: "Process payment between agents",
        input_schema: {...}
      },
      {
        name: "locus_transfer_funds",
        description: "Transfer funds to agent wallet",
        input_schema: {...}
      },
      
      // Proxied Consulate tools
      {
        name: "locus_file_dispute",
        description: "File payment dispute (powered by Consulate)",
        input_schema: {
          type: "object",
          properties: {
            paymentId: {
              type: "string",
              description: "Locus payment ID"
            },
            reason: {
              type: "string",
              description: "Reason for dispute"
            },
            amount: {
              type: "number",
              description: "Amount in dispute"
            },
            merchantDid: {
              type: "string",
              description: "Merchant's agent DID"
            }
          },
          required: ["paymentId", "reason", "amount", "merchantDid"]
        }
      },
      {
        name: "locus_check_dispute",
        description: "Check dispute status (powered by Consulate)",
        input_schema: {
          type: "object",
          properties: {
            caseId: {
              type: "string",
              description: "Consulate case ID"
            }
          },
          required: ["caseId"]
        }
      }
    ];
    ```
    
    ### Implementation
    
    ```typescript
    export const mcpInvoke = async (ctx, request) => {
      const { tool, parameters } = await request.json();
      
      // Handle native tools
      if (tool === 'locus_process_payment') {
        return await processPayment(ctx, parameters);
      }
      
      // Proxy dispute tools to Consulate
      if (tool === 'locus_file_dispute') {
        // Map Locus parameters to Consulate format
        const consulateParams = {
          plaintiff: parameters.buyerDid || extractFromPayment(parameters.paymentId),
          defendant: parameters.merchantDid,
          disputeType: 'PAYMENT_FAILURE',
          claim: `Payment ${parameters.paymentId} failed: ${parameters.reason}`,
          claimAmount: parameters.amount,
          jurisdiction: 'US'
        };
        
        // Forward to Consulate with your partner API key
        const response = await fetch('https://api.consulatehq.com/mcp/invoke', {
          method: 'POST',
          headers: {
            'Authorization': `Bearer ${process.env.LOCUS_CONSULATE_API_KEY}`,
            'Content-Type': 'application/json'
          },
          body: JSON.stringify({
            tool: 'file_dispute',
            parameters: consulateParams
          })
        });
        
        const result = await response.json();
        
        // Map back to your format
        return new Response(JSON.stringify({
          success: result.success,
          disputeId: result.caseId,
          trackingUrl: result.trackingUrl,
          locusPaymentId: parameters.paymentId,
          estimatedResolution: result.estimatedResolution
        }), {
          headers: { 'Content-Type': 'application/json' }
        });
      }
      
      if (tool === 'locus_check_dispute') {
        // Simple passthrough
        return await fetch('https://api.consulatehq.com/mcp/invoke', {
          method: 'POST',
          headers: {
            'Authorization': `Bearer ${process.env.LOCUS_CONSULATE_API_KEY}`,
            'Content-Type': 'application/json'
          },
          body: JSON.stringify({
            tool: 'check_case_status',
            parameters: { caseId: parameters.caseId }
          })
        });
      }
    };
    ```
    
    ### User Experience
    
    ```javascript Agent Using Locus
    // Agent only connects to Locus
    const payment = await mcp.invoke('locus_process_payment', {
      from: 'wallet_123',
      to: 'wallet_456',
      amount: 5000
    });
    
    if (!payment.success) {
      // File dispute through Locus (Consulate under the hood)
      const dispute = await mcp.invoke('locus_file_dispute', {
        paymentId: payment.paymentId,
        reason: 'Goods not delivered',
        amount: 5000,
        merchantDid: 'did:agent:merchant-xyz'
      });
      
      console.log(`Dispute filed: ${dispute.disputeId}`);
    }
    ```
    
    ### Benefits
    - ✅ Users only need ONE API key (yours)
    - ✅ Branded experience (`locus_file_dispute` not `consulate_file_dispute`)
    - ✅ You can add platform-specific context (payment IDs, etc.)
    - ✅ You control when/how disputes are surfaced
    
    ### Requirements
    - ❌ You need to maintain Consulate API key
    - ❌ Add ~50 lines of proxy code per tool
    - ❌ Update if Consulate changes parameters
  </Tab>

## Complete Integration Example: Locus

### Scenario
Locus processes $10M/month in agent-to-agent payments. When payments fail, buyers need to file disputes.

### Option A: Proxy Pattern (Recommended)

```typescript locus-backend/mcp.ts
import { httpAction } from "./_generated/server";

const CONSULATE_API_KEY = process.env.LOCUS_CONSULATE_API_KEY;
const CONSULATE_MCP_URL = 'https://api.consulatehq.com/mcp/invoke';

export const MCP_TOOLS = [
  // Native Locus tools
  {
    name: "locus_process_payment",
    description: "Process payment between agents",
    input_schema: {
      type: "object",
      properties: {
        from: { type: "string", description: "Sender wallet ID" },
        to: { type: "string", description: "Recipient wallet ID" },
        amount: { type: "number", description: "Amount in USD" }
      },
      required: ["from", "to", "amount"]
    }
  },
  
  // Dispute tools (proxied to Consulate)
  {
    name: "locus_file_payment_dispute",
    description: "File dispute for failed payment or chargeback",
    input_schema: {
      type: "object",
      properties: {
        paymentId: { type: "string", description: "Locus payment ID" },
        disputeReason: { 
          type: "string", 
          enum: ["goods_not_delivered", "service_not_provided", "quality_issue", "unauthorized_charge"]
        },
        amount: { type: "number", description: "Amount to dispute" },
        evidence: { 
          type: "object", 
          description: "Evidence (receipts, messages, etc.)" 
        }
      },
      required: ["paymentId", "disputeReason", "amount"]
    }
  },
  {
    name: "locus_check_dispute_status",
    description: "Check status of payment dispute",
    input_schema: {
      type: "object",
      properties: {
        disputeId: { type: "string", description: "Consulate case ID" }
      },
      required: ["disputeId"]
    }
  }
];

export const mcpInvoke = httpAction(async (ctx, request) => {
  const { tool, parameters } = await request.json();
  
  // Authenticate Locus user
  const locusAuth = await authenticateLocusUser(request.headers);
  if (!locusAuth.valid) {
    return new Response(JSON.stringify({ error: "Unauthorized" }), {
      status: 401,
      headers: { 'Content-Type': 'application/json' }
    });
  }
  
  // Handle native tools
  if (tool === 'locus_process_payment') {
    return await handlePayment(ctx, parameters, locusAuth.userId);
  }
  
  // Proxy dispute tools to Consulate
  if (tool === 'locus_file_payment_dispute') {
    // Get payment details from your DB
    const payment = await ctx.db
      .query("payments")
      .filter(q => q.eq(q.field("paymentId"), parameters.paymentId))
      .first();
    
    if (!payment) {
      return new Response(JSON.stringify({
        error: "Payment not found"
      }), { status: 404 });
    }
    
    // Upload evidence to your storage
    const evidenceUrl = await uploadEvidence(
      parameters.paymentId,
      parameters.evidence
    );
    
    // Submit evidence to Consulate first
    await fetch(CONSULATE_MCP_URL, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${CONSULATE_API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        tool: 'submit_evidence',
        parameters: {
          agentDid: payment.buyerDid,
          sha256: hashEvidence(parameters.evidence),
          evidenceUrl: evidenceUrl
        }
      })
    });
    
    // File dispute with Consulate
    const consulateResponse = await fetch(CONSULATE_MCP_URL, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${CONSULATE_API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        tool: 'file_dispute',
        parameters: {
          plaintiff: payment.buyerDid,
          defendant: payment.merchantDid,
          disputeType: 'PAYMENT_FAILURE',
          claim: `Locus payment ${parameters.paymentId} - ${parameters.disputeReason}`,
          claimAmount: parameters.amount,
          jurisdiction: payment.jurisdiction || 'US'
        }
      })
    });
    
    const consulateResult = await consulateResponse.json();
    
    // Store case ID in your DB for tracking
    await ctx.db.patch(payment._id, {
      disputeId: consulateResult.caseId,
      disputeStatus: 'filed',
      disputeFiledAt: Date.now()
    });
    
    // Return Locus-branded response
    return new Response(JSON.stringify({
      success: true,
      paymentId: parameters.paymentId,
      disputeId: consulateResult.caseId,
      status: 'filed',
      message: 'Payment dispute filed successfully',
      trackingUrl: `https://paywithlocus.com/disputes/${consulateResult.caseId}`,
      consulateUrl: consulateResult.trackingUrl,
      estimatedResolution: consulateResult.estimatedResolution
    }), {
      headers: { 'Content-Type': 'application/json' }
    });
  }
  
  if (tool === 'locus_check_dispute_status') {
    // Passthrough to Consulate
    const response = await fetch(CONSULATE_MCP_URL, {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${CONSULATE_API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        tool: 'check_case_status',
        parameters: { caseId: parameters.disputeId }
      })
    });
    
    return response;
  }
});
```
    
    **User Flow:**
    ```
    1. Agent processes payment via Locus
       ↓
    2. Payment fails
       ↓
    3. Agent invokes: locus_file_payment_dispute
       ↓
    4. Locus proxies to Consulate (transparent to agent)
       ↓
    5. Dispute filed, agent gets Locus-branded response
       ↓
    6. Agent monitors via: locus_check_dispute_status
    ```

## Tools You Should Proxy

### Essential (Proxy These)
1. **`file_dispute`** → Your branded name (e.g., `locus_file_payment_dispute`)
2. **`check_case_status`** → Your branded name (e.g., `locus_check_dispute`)

4. `lookup_agent` - Most platforms already know both parties

6. `get_sla_status` - Only if you track SLA compliance

7. `register_agent` - Users should register directly with Consulate
