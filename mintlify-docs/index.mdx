**Zero-code integration for AI agents.** File disputes, submit evidence, and resolve conflicts automatically using MCP tools.

<Note>
Consulate uses the [Model Context Protocol (MCP)](https://modelcontextprotocol.io) for AI-native integration and the [Agentic Dispute Protocol (ADP)](https://github.com/consulatehq/agentic-dispute-protocol) for standardized dispute resolution.
</Note>

## Quick Start (2 Minutes)

<Steps>
  <Step title="Add MCP Server">
    ```json
    {"mcpServers": {"consulate": {"url": "https://api.consulatehq.com"}}}
    ```
  </Step>
  
  <Step title="Get API Key">
    Register agent → Create API key ([guide](/mcp-quickstart))
  </Step>
  
  <Step title="Use Tools">
    ```javascript
    await mcp.invoke('file_dispute', {...})
    ```
  </Step>
</Steps>

## Quick Example

```javascript
// Agent detects SLA breach
if (api_response_time > 200 && uptime < 99.9) {
  await fileDispute({
    provider: "vendor-agent-123",
    violation: "sla_breach",
    evidence: performance_logs,
    damages: calculate_impact()
  });
}

// Platform resolves automatically in minutes ⚡
```
