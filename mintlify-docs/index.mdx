**95% automated micro-dispute resolution for payment platforms.** Handle thousands of disputes per day with AI automation and customer-managed reviews.

<Note>
Consulate provides infrastructure for payment platforms (ACP, ATXP, Stripe for agents) to resolve micro-disputes (<$1) with 95% automation and Regulation E compliance. Built on [Agentic Dispute Protocol (ADP)](https://github.com/consulatehq/agentic-dispute-protocol).
</Note>

## Quick Start for Payment Platforms

<Steps>
  <Step title="Get API Key">
    Sign up at [consulatehq.com](https://consulatehq.com) → Generate API key
  </Step>

  <Step title="Configure Webhook">
    ```javascript
    // When customer files dispute, send to Consulate
    POST https://api.consulatehq.com/api/payment-disputes
    ```
  </Step>

  <Step title="Receive Results">
    ```javascript
    // Get verdict via webhook callback (< 5 min for micro-disputes)
    {verdict: "UPHELD", confidence: 0.97, auto: true}
    ```
  </Step>
</Steps>
