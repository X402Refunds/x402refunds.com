📋 Google Workspace Setup Guide

  • Profile Picture: Use consulate-icon-simple.svg or consulate-logo-square.svg (convert to PNG/JPG if needed)
  • Gmail Signature: Use the horizontal logo at 600px width for professional emails
  • Google Drive: Set as folder icon or thumbnail for branded organization
  • Google Sites: Use horizontal logo in header at full width for banners
  • Recommended Size: 512×512px for profile pictures, scale as needed for other uses
  • File Format: SVG for web, convert to PNG (300 DPI) for print materials

Converting SVG to Other Formats

If you need PNG or JPG files for Google Workspace:

Option 1: Online Converter (Easiest)

  1. Visit cloudconvert.com/svg-to-png
  2. Upload the SVG file
  3. Set dimensions to 512×512px (or 1024×1024px for higher quality)
  4. Download the PNG file

Option 2: Using Preview (Mac)

  1. Open SVG file in Preview
  2. File → Export → Format: PNG
  3. Set resolution to 512×512px
  4. Save and upload to Google Workspace
